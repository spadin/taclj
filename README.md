# Taclj

## Front-end interface to [Ticlj, a command-line Tic Tac Toe game][1]

You can [view the Taclj web app][2] on Heroku.

Running specs

    $ lein spec

Start the Taclj server

    $ lein joodo server


[1]: http://github.com/spadin/ticlj
[2]: http://taclj.herokuapp.com
